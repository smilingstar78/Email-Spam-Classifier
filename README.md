# 📧 Email Spam Classifier

## 🚀 Overview  
This project is a **Machine Learning-based Email Spam Classifier** that detects whether an email is **spam or not spam (ham)**. It uses Natural Language Processing (NLP) techniques to analyze email text and make predictions, helping filter unwanted messages and improve inbox management.

---

## 🎯 Features  
- Classifies emails as **Spam** or **Not Spam**  
- Uses **text preprocessing** techniques  
- Implements **machine learning algorithms**  
- Beginner-friendly and easy to understand  
- Can be extended into a real-world application  

---

## 🧠 How It Works  

1. **Data Collection**  
   A dataset containing labeled emails (spam/ham) is used  

2. **Data Preprocessing**  
   - Convert text to lowercase  
   - Remove punctuation and stopwords  
   - Tokenization  

3. **Feature Extraction**  
   - Convert text into numerical form using:
     - Bag of Words  
     - TF-IDF  

4. **Model Training**  
   - Train a machine learning model (e.g., Naive Bayes, Logistic Regression)  

5. **Prediction**  
   - Model predicts whether a new email is spam or not
